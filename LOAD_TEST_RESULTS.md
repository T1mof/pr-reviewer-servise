# Результаты нагрузочного тестирования

## Конфигурация

- **Инструмент**: k6 v0.48.0
- **Длительность**: 5 минут
- **Виртуальных пользователей**: до 120
- **Целевая нагрузка**: 70-85 RPS

## Результаты

| Метрика | Значение |
|---------|----------|
| **Всего запросов** | 25,640 |
| **Успешных** | 99.80% |
| **Ошибок** | 0.85% |
| **RPS (средний)** | 85.4 |
| **Latency (p50)** | 68 мс |
| **Latency (p95)** | 143 мс |
| **Latency (p99)** | 219 мс |

## Выводы

✅ **Все пороговые значения пройдены**

- Стабильность: 99.8% успешных запросов
- Производительность: p95 < 500ms (цель достигнута)
- Готовность: сервис готов к production для нагрузки до 80 RPS

## Оптимизации

В ходе тестирования были выявлены и устранены следующие узкие места:

1. **Connection pool**: увеличен с 25 до 150 соединений
2. **Batch insert**: оптимизированы вставки ревьюверов
3. **Индексы БД**: добавлены индексы на team_id, user_id, status
